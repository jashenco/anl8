# Fit+ System

# Setup

- Create Virtual Environment
```
virtualenv venv
```

- Activate Virtual Environment
```
.\venv\Script\activate
```

- Install Dependecies
```
pip install -r requirements.txt
```

- Run main script
```
py main.py
```

# Working together

- When adding dependencies
```
pip freeze > requirements.txt
```

- Work as much OPP as possible
